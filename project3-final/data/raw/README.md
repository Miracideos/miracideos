# Dados brutos

Os arquivos FASTQ brutos e os arquivos intermediarios pesados de quantificacao nao foram incluidos no repositorio por tamanho.

Os dados originais foram obtidos de runs publicos de RNA-seq do estudo SRP035609. A analise final usa as amostras das geracoes 2 e 3, apos exclusao da geracao 1 por criterio de qualidade.
